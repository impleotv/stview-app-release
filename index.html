Setup

Install instructions

Windows

  1. Download the Windows MSI package.
  2. Open the installer and follow the prompts.
  3. Launch STView from the Start menu or desktop shortcut.

macOS

  1. Download the macOS DMG package when available.
  2. Open the DMG and drag STView into Applications.
  3. The macOS package is currently unsigned.
  4. If Gatekeeper blocks the first launch, right-click StInspector.app, select Open, and confirm the prompt.
  5. If macOS still blocks the app, allow it once in System Settings > Privacy & Security, then launch it again.

Linux

  1. Download the Linux DEB package when available.
  2. Install it with sudo apt install ./STView-...linux-amd64.deb.
  3. If using dpkg -i, run sudo apt -f install to repair missing dependencies.
